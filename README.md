Uso por otros usuarios

Dado que el buscador presenta dos elementos que requieren acceso (el buscador y la base de datos de búsqueda -excel-), es necesario ajustar estos elementos para que otros usuarios lo puedan usar. Inicialmente, es necesario reemplazar la aplicación, por lo cual se abre power apps make.powerapps.com y se va a la sección “Aplicaciones”



Allí se va a “Importar aplicación”



En la parte de “Acción” (donde hay una llave), se selecciona y cambia la configuración a “Crear una nueva”


Ahora es necesario crear un excel en la nube, el cual debe contener una tabla (Insertar > Nueva tabla) con una columna llamada “TextoBuscado”. Luego ingresar a la aplicación recién creada y en la solicitud de permiso cambiar la conexión al nuevo excel creado.




Posteriormente, en Power BI es necesario eliminar el buscador y volverlo a introducir ahora desde la aplicación de su cuenta.
